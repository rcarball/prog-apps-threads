Creación y configuración básica de un JFrame

En Java Swing, JFrame es la clase fundamental para crear una ventana en la interfaz
gráfica de una aplicación. La configuración de un JFrame implica varios pasos 
básicos, cómo establecer su tamaño, título y visibilidad. Para empezar, se crea 
una instancia de JFrame, generalmente con un título que se muestra en la barra 
superior. Luego, se configuran propiedades como el tamaño de la ventana mediante 
setSize() y su visibilidad con setVisible(true). También es importante definir el 
comportamiento al cerrar la ventana, utilizando setDefaultCloseOperation() para 
especificar si la aplicación debe finalizar o simplemente ocultar la ventana. 
Otro aspecto crucial en la configuración de un JFrame es la asignación de un 
layout adecuado, que organizará los componentes visuales dentro de la ventana. 
Una vez configurado, el JFrame puede contener diversos componentes, como botones, 
etiquetas o paneles, creando así una interfaz completa para el usuario. Esta 
clase proporciona la base sobre la cual se construyen ventanas interactivas y 
atractivas en aplicaciones Swing.
